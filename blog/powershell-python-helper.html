PowerShell $PROFILE — Python helper

22. října 2025 | powershell python

Python a virtuální prostředí — věčné opakování stejných příkazů. python -m venv .venv, pak aktivace, pak spuštění. Tři kroky, které děláš u každého projektu. Řešení? Tři funkce v PowerShell $PROFILE.

pi (Invoke-InitPython) založí venv, pa (Invoke-ActivatePython) ho aktivuje, pr (Invoke-RunPython) spustí skript i s parametry. Žádné googlení "jak aktivovat venv na Windows". Jeden profil, tři aliasy, klid.
